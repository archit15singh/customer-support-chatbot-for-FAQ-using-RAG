How to make images accessible for people Skip to main content Help Center How to make images accessible for people When you post photos using the X app for iOS or Android, or on X.com, you have the option to compose a description of the images so the content is accessible to more people, including those who are blind or low-vision.Good image descriptions are concise and descriptive, helping people understand whatâs happening in an image.This article includes instructions for composing image descriptions using: How to set image descriptions How to add image descriptions in posts from X for iOS Step 1 Start by tapping the Post compose icon and attach your photo(s).Note : For detailed instructions about adding photos to your posts, read this article.Step 2 On the image, tap the +Alt button to insert descriptive text.Step 3 Type your description of the image and tap Done.Tap the description again to edit it prior to posting the post.(The limit is 1000 characters.) Step 4 You can add a description to each image in a post.Note : Image descriptions cannot be added to videos.How to add image descriptions in posts from X for Android Step 1 Start by tapping the Post icon and attach your photo(s).Note : For detailed instructions about adding photos to your posts, read this article.Step 2 On the image, tap the +Alt button to insert descriptive text.Step 3 Type your description of the image and tap Done.Tap the description again to edit it prior to posting the post.(The limit is 1000 characters.) Step 4 You can add a description to each image in a post.Note : Image descriptions cannot be added to videos.How to add image descriptions in posts from x.com Step 1 Click on the Post compose button, or press the ânâ key to use the keyboard shortcut.Step 2 Attach your photo(s).Note : For detailed instructions about adding photos to your posts, read this article.Step 3 To insert descriptive text, click Add description.Step 4 Type your description of the image and click the Done button.To edit the description, re-open the Add description dialog prior to posting the post.(The limit is 1000 characters.) Step 5 You can add a description to each image in a post.Note : Image descriptions cannot be added to videos.VoiceOver on X for iOS How to add image descriptions in posts using VoiceOver Step 1 Find the Compose post button at the top right of the screen using touch exploration.Step 2 Double-tap the Compose post button.Step 3 Find the Photo library button using either touch exploration or the right/left flick gesture.Step 4 Double- tap the Photo library button to add an image.Step 5 Use touch exploration or the right/left flick gesture to find the image to add.Step 6 Double-tap an image to add it and return to the composer.Step 7 In the composer, find the image using touch exploration or the right/left flick gesture.Step 8 While focused on the image, swipe up or down until you hear " Add description to photo " and then double-tap to add a description for the image.Step 9 Type your description of the image.(The limit is 1000 characters.) Step 10 Find the Done button in the top right of the screen using touch exploration or the right/left flick gesture.Step 11 Double-tap the Done button to add the description and return to the composer.How to edit image descriptions in posts using VoiceOver Step 1 You can edit your image description prior to posting the post.In the composer, find the image using touch exploration or the right/left flick gesture.Step 2 While focused on the image, swipe up or down until you hear " Edit description for photo " and then double-tap to add a description to the image.Step 3 Change the description of the image.(The limit is 1000 characters.) Step 4 Find the Done button in the top right of the screen using touch exploration or the right/left flick gesture.Step 5 Double-tap the Done button to add the description and return to the composer.How to add image descriptions in posts using TalkBack Step 1 Find the Write button at the bottom left of the screen using touch exploration.Step 2 Double-tap the Write button.Step 3 Find the Photos button using either touch exploration or the right/left flick gesture.Step 4 Double-tap the Photos button to add a photo.Step 5 Use touch exploration or the right/left flick gesture to find the photo to add.Step 6 Double-tap a photo to add it and return to the composer.Step 7 In the composer, find the Add description button using touch exploration or the right/left flick gesture.Step 8 Double-tap the Add description button.Step 9 Type your description of the image.(The limit is 1000 characters.) Step 10 Find the Done button in the top right of the screen using touch exploration or the right/left flick gesture.Step 11 Double-tap the Done button to add the description and return to the composer.( Note: after adding the description, the label of the Add description button will be changed to match the description.) How to edit image descriptions in posts using TalkBack Step 1 You can edit your image description prior to posting the post.Find the button labeled with the current description using touch exploration or the right/left flick gesture.Step 2 Double-tap the button.Step 3 Change the description of the image.(The limit is 1000 characters.) Step 4 Find the Done button in the top right of the screen using touch exploration or the right/left flick gesture.Step 5 Double-tap the Done button to add the description and return to the composer.( Note: after adding the description, the label of the Add description button will be changed to match the description.)
